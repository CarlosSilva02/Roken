# Roken
Secure student marketplace for UTRGV
---

#Features

- User account system (login, register, verified UTRGV emails)
- Post items with photos, descriptions, and prices
- Search and filter listings
- Built-in messaging between buyers and sellers
- Social features: profiles, interests, and item history
- Modern UI designed in Figma

---

# Tech Stack Frontend

- HTML / CSS / JavaScript   
- React



# Database Structure

- Users
- Listings
- Messages
- Favorites
- Transactions
